# 🚀 Εγκατάσταση για Διαχειριστή - ΑΠΛΗ ΕΚΔΟΧΗ

## Μόνο 2 Βήματα

### Βήμα 0️⃣ - ΚΑΤΕΒΑΣΜΑ (αν δεν το έχεις)

1. Ο κύριος διαχειριστής σας δίνει το link του repository
2. Πάτησε το πράσινο κουμπί `Code` → `Download ZIP`
3. Ξεκόμπρεσε το αρχείο σε κάποιο φάκελο (π.χ. `C:\Giatros\`)

### Βήμα 1️⃣ - SETUP (Μια φορά)

1. Άνοιξε τον φάκελο
2. **Διπλοκλικ**: `SETUP.bat`
3. Περίμενε να τελειώσει
4. Πάτησε Enter

### Βήμα 2️⃣ - START (Κάθε φορά που θέλεις)

**Διπλοκλικ**: `START-AGENT.bat`

Έτοιμο! Ο Agent τρέχει. 📡

---

## 📊 Δες τα Logs

Άνοιξε στο browser:
```
http://localhost:4000
```

Εκεί θα δεις:
- ✅ Παραγγελίες που εκτυπώθησαν
- ❌ Σφάλματα (αν υπάρχουν)
- 📡 Κατάσταση σύνδεσης

---

## ⚠️ Σημαντικό

✅ Ο εκτυπωτής πρέπει να είναι ενεργός  
✅ Ο Agent πρέπει να τρέχει πάντα (ακόμα και νύχτα)  
✅ Το `START-AGENT.bat` θα "παγώσει" όσο τρέχει - αυτό είναι φυσιολογικό  

---

## 🔧 Αν θέλεις να αλλάξεις ρυθμίσεις

Ρύθμιση εκτυπωτή ή άλλα: Άνοιξε το `.env` με Notepad

```
CLOUD_API_URL=https://giatroslive.onrender.com
LOCAL_PRINTER_HOST=192.168.88.4
LOCAL_PRINTER_PORT=9100
POLL_INTERVAL_SECONDS=10
STATUS_PORT=4000
```

Αλλάξε τι χρειάζεται και σώσε.

---

## ❌ Προβλήματα

### Δεν εκτυπώνει
- Ελέγχει ότι ο εκτυπωτής είναι ενεργός
- Δες τα logs: http://localhost:4000

### Δεν ξεκινάει το SETUP.bat
- Ίσως λείπει το Node.js
- Κατέβασε από: https://nodejs.org/

### CLOUD_API_URL error
- Ζητά από τον κύριο διαχειριστή το σωστό URL
